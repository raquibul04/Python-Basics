# Brand Name Generator

A simple Python program that generates a brand name based on your city and pet's name.

## Description

This program asks the user for:

1. The city they grew up in.
2. Their pet's name.

It then combines these inputs to suggest a potential brand name.

## How to Use

1. Make sure you have Python installed on your computer.
2. Clone or download this repository.
3. Run the program:

```bash
python brand_name_generator.py
